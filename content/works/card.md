overview:
  - 進級展で企業の方に自分のことを知っていただくために名刺を制作しました。
  - また、企業の方との挨拶の際に名前を覚えていただきやすくすることを目的としています。
  - 進級展での使用をきっかけに、今後も企業の方とお会いする場面で継続して活用できるよう、実用性を意識して制作しました。

process:
  - title: 調査（参考探し）
    type: hearing
    lead: まず、デザインの参考にするためにインターネットで様々な名刺デザインを調べました。特に、配色やデザインのベース、文字の配置などを意識して参考にしました。また、フロントエンドエンジニアを目指す者として、どのようなデザインが合うのかについても調べました。

  - title: デザイン
    type: design
    lead: 調査した結果、フロントエンドエンジニアのイメージに合わせ、黒をベースにしたシンプルなデザインにしました。また、高級感を出すために線を用いたデザインを取り入れ、落ち着いた印象になるよう工夫しました。さらに、名前を覚えてもらいやすくするため、名前を目立たせたレイアウトにしています。
    images:
      - /image/name-card_table.webp
      - /image/name-card_back.webp
    desc:
      - 表面には名前だけでなく、連絡をしてもらえるようメールアドレスを記載しました。また、エンジニアとして制作物を確認してもらえるよう、GitHubのURLも掲載しています。
      - 裏面は、表面とは対照的に白をベースとしたデザインにしました。また、自分のブログへアクセスできるようQRコードとURLを掲載しています。さらに、志望職種を記載し、自分がどのような分野を目指しているのかが伝わるようにしました。
      - デザインは表面と統一感が出るよう、基本的なレイアウトを揃えています。

  - type: coding
    show : false

  - title: 印刷
    type: adjustment
    text:
      - デザイン完成後、ラクスルに依頼して名刺を印刷しました。
      - 用紙は学校にある見本を実際に触って確認しながら、デザインに合うものを選ぶようにしました。また、印刷時に文字が潰れてしまわないよう、文字の読みやすさも考えました。
      - その結果、和風紙を使用し、落ち着いた質感と高級感のある仕上がりになるようにしました。

ingenuity:
  - title: 情報配置
    type: update
    blocks:
    - text: 表面は、相手がすぐに自分の名前を確認できるよう、名前を最も大きくし、中央に配置しました。学校名や連絡先などの情報は、小さすぎず、かつ主張しすぎないようバランスを意識して配置しています。裏面は、自分のブログを見てもらえるよう、URLを上部に控えめに配置し、QRコードを中央に大きく配置しました。その下には志望職種を記載し、自分の方向性が伝わるように工夫しました。

  - title: 情報設計
    type: update
    blocks:
    - text: もう一つの工夫として、URLや連絡先の横にアイコンを配置しました。視覚的に情報の種類が分かりやすくなるようにすることで、ひと目で内容を認識できるよう工夫しています。

reflection:
  - この作品を通して、名刺は自分を見せるだけでなく、受け取る相手の視点で考えることも大切だと感じました。
  - また、名前や連絡先などの情報の優先順位を考えて配置することの重要性も学びました。
  - さらに、名刺は自分のイメージや目指している職種を伝えるツールでもあると感じました。
  - 配色やレイアウトによって印象が変わるため、イメージに合ったデザインを考えることの大切さも学びました。
